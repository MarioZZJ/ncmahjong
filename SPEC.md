# 南昌麻将记分器 - 规格文档

## 项目概述
- **项目名称**: 南昌麻将记分器
- **类型**: 移动端网页应用 (PWA)
- **核心功能**: 计算南昌麻将胡牌番数及分数
- **目标用户**: 南昌麻将玩家

## 番数规则

### 基本番数
| 牌型 | 番数 |
|------|------|
| 屁胡（基本胡） | 1番 |
| 门前清 | 1番 |
| 碰碰胡 | 2番 |
| 混一色 | 3番 |
| 清一色 | 4番 |
| 七对 | 3番 |
| 豪华七对 | 4番 |
| 海底捞月 | 2番 |
| 杠上开花 | 2番 |
| 抢杠胡 | 2番 |
| 全求人 | 2番 |

### 特殊规则
- **庄家**: 1番
- **硬赖子**: 每个+1番
- **软赖子**: 不另计番

### 分数计算
- 基础分 = 2^(番数-1) × 底分(通常为1)
- 庄家胡：×2
- 闲家胡庄家输：×2

## UI/UX 规格

### 布局
- 单页应用，移动优先
- 竖屏模式最佳
- 底部固定操作栏

### 视觉
- 主色: #1a73e8 (蓝)
- 背景: #f5f5f5
- 卡片: #ffffff
- 文字: #333333
- 强调: #ff6b6b (红)

### 交互
- 触摸友好的大按钮
- 滑动选择数字
- 即时计算显示结果

## 验收标准
1. ✅ 可选择胡牌类型（多选）
2. ✅ 可设置赖子数量（0-4）
3. ✅ 可设置庄家/闲家
4. ✅ 实时显示计算结果
5. ✅ 移动端响应式布局
6. ✅ 可部署到GitHub Pages
